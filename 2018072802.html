配置nexT主题

一、部署nexT主题

  首先在项目根目录，打开命令行，执行以下指令，将主题部署到项目内。

$ git clone https://github.com/iissnan/hexo-theme-next themes/next

  打开根目录配置文件，将主题换为next。

theme: next

  本地查看一下效果，发现4000端口被占用，查看端口情况:

netstat -o -n -a | findstr :4000

  干掉占用的进程。

taskkill /F /PID 7532

  或者指定其他端口。

$ hexo s -p 5000

  部署后查看以下效果
