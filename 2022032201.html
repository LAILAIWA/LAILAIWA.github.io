从零开启的跑路规划，包括短期和长期计划，内容主要为生活和工作上时间分配、周期计划、雅思准备等。

润！菜鸟英语从零开始😭

一. 前文

1.1 动机

理由：

  • 国内IT行业工作强度大，加班甚至无偿加班现象越来越多。
  • 经济不景气，且看不到未来有好转的迹象。
  • 疫情，没完没了的隔离和核酸检测。
  • 社会氛围一年不如一年，言论管控上升到前所未有的力度。
  • 身体越来越差，今年胃病还严重了。
  • 和周围人割裂感愈发严重，也许是时候换个环境了。
  • 掌握一门第二语言。
  • 上海大吃饱。— 20220405。

难点：

  • 本科没有刷分，申不到好学校。
  • 工作不能暂停，还需要腾出时间学习技术，以及刷题。
  • 身体不如以前，不能熬夜。
  • 英语基础差。
  • 注意力不集中。
  • 记忆力较差。

1.2 目标

  • 首先既然申不到好学校，留学的性价比不高。且时间成本太大，加上已有的三到四年工作经验，暂时排除掉留学的选择。如果在30岁前没有能够出去，且还是很想出去的话，可以再选择这条相对容易的道路。（— 20220409：综合考虑还是留学比较合适）
  • 还有的选择就是直接出国工作，但在技术和语言双双未达标的情况下，仍需要定一个比较现实或者能够阶段性到达的目标。
    • 在国内进一家外企等待交换机会。
      • 优点：没有工作空档期。
      • 缺点：机会不一定很高，有很大失败几率，时间成本也较大。
    • Linkedin直接面，先争取到加拿大公司的Offer，解决身份问题，攒积分拿到绿卡。
      • 优点：如果这条可以走的话，也没有太大的工作空档期，且时间成本较小。
      • 缺点：需要继续了解相关工作机会，是否有足够的公司有能力帮助解决身份问题。可能对口语或技术要求较高。

直接工作移民这种渠道，需要雇主担保，会因为保护当地人就业政策不一定是个容易实现的方式，小公司的话相比海外招人应该直接本地录取成本较小，大公司的话不知道是不是更倾向于full remote。需要继续收集相关信息，最好技术能够达到直接进大厂水平。

二. 计划

2.1 阶段一

周期：4月~6月

  • 补完Redis。
  • 补完分布式基础。
  • 雅思前期准备。
  • 背单词：扇贝雅思1500 + 538雅思考点词。每天20词（工作日），周末50词。

2.2 阶段二

周期：2022年6月到2023年1月

  • 继续背单词。
  • 雅思学习和刷题。
  • LeetCode刷题。

2.3 阶段三

周期：2023年1月~4月 或 5月到8月 或 9月到12月

  • 考雅思。
  • 准备材料。

2.4 阶段四

周期：2023年到2025年

  • 如果阶段3进展不顺，继续考试和准备材料。
  • 如果有书读，做好相关准备。
  • 刷LeetCode。

三. QA

3.1 签证类别

（常见）

  • 短期签证：

    • 访问签证：最长6个月。

      包括：旅游、探亲和陪读等。

      疫情期间国内过海关可能需要返程机票，旅游签等通过几率没有那么大（非必要不出境）。

  • 长期签证：

    • 学习签证：时间和学期周期相同。

      （可以先申请旅签，境内转学签；或者准备二次尝试，建议后者）

    • 工作签证：

      • 指定雇主：LMIA
      • 开放工签：包括毕业后工签、配偶工签、实习工签和桥梁工签等

  • 永久签证：

    • 枫叶卡。

3.2 签证流程

  1. 收到录取通知书后撰写学习计划（重要考察依据）。
  2. 资金担保，需要注意一些事项。
  3. 填写签证申请表，可以找专业机构帮忙。
  4. 到加拿大使馆指定地点体检。需要准备：使馆邮寄的体检表、身份证、三张两寸照片。
  5. 将材料递交给签证中心。所需材料：
    • 护照：有效期要在回程日期算9个月以上。
    • 照片：近六个月，1张，35*45。
    • 身份证：A4纸复印正反两面。
    • 户口本：原件。
    • 录取通知书：复印件，显示学费金额，学期起止时间，最迟可注册时间等。
    • 等等。

可参考：加拿大签证办理_加拿大签证办理流程 - 加拿大签证中心网站 (jianada-qianzheng.com)，或者去相关官方网站查阅信息。

3.4 如何选择国家？

  • 美国：大部分人的最终目的地，就业机会最多。
  • 加拿大：薪水相同级别可能是美国的一半，高税，工作机会相对较少，气候寒冷，很多城市可能比较村一些。绿卡比较容易拿到，比较好留下，所以可以作为移民首选。
  • 新加坡：
  • 泰国：可以先走旅行签证（概率低），商务签（成本3W左右），再办工签2W，必须有泰国中介公司配合。
  • 澳洲：类似加拿大的积分策略，但疫情期间政策不太好进。

3.X 需要准备多少资金？

申请签证时需要的账目资金：没有具体留学担保要求，需要综合考虑学费和读书地点。需要注意收入的合法性。

3.X 如何在线申请签证？

加拿大在线申请地址：

  • 旅行豁免：How to apply online for a visitor visa - Canada.ca

    需要枫叶卡或有卡人的直系亲属。

  • 学签：Study permit: How to apply - Canada.ca

    材料：

    • 留学签证申请表。
    • 有效护照：有效期需在6个月以上。
    • 身份证。
    • 照片。
    • 户口本。
    • 学校录取 offer。
    • 已缴学费、注册费等费用收据。
    • 学习计划书。
    • 最高学历毕业证书及学位证书。
    • 雅思/托福成绩。
    • 体检报告。
    • 无犯罪记录证明。
    • 经济证明，如存款证明。

  • 工签：Application to work in Canada — work permits including open work permits - Canada.ca

在加拿大境内可以在未过期时在线申请延长逗留时间。
