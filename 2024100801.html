血脂

需要严格注意血脂水平的人群（每年定期体检）：

  1. 冠心病患者
  2. 发生过心梗
  3. 经历短暂性脑缺血
  4. 外周血管有动脉粥样硬化
  5. 糖尿病患者
  6. 男性>45，女性>55
  7. 吸烟、肥胖人群

控制血脂：

  1. 调整生活方式，健康饮食
  2. 控制体重
  3. 戒烟戒酒
  4. 适当运动
  5. 药物治疗：常用有化学他汀类药品
  6. 定期检查肝功能和激酶

心梗

急性心肌梗死，急性心肌缺血性坏死，在冠状动脉病变基础上发生了冠脉的供血急剧减少或中断，最终导致心肌细胞死亡。伴随着心律失常、休克、心力衰竭。较为严重的心梗只有6分钟的黄金抢救时间，症状轻者时间会略微长一些，超过6分钟，停止脑部供血，中枢神经会遭受不可逆的损伤。

家用12导心电仪（多导联，手表一般是单导联）。

  • 症状：牙痛，胸口被石头压。身体局部疼痛（胸、牙、胃、胳膊、背），心慌、呼吸困难、面色苍白、出冷汗、有濒死感。识别方法：心电图，ST段压低、倒置、弓背抬高。恶性心律失常，伴有异常Q波等。
  • 高危人群：心脏手术过，患有心脏的基础病变如心律失常、心肌缺血/坏死、心脏器质性病变等。三高，特别是控制不佳的高血压。
  • 处理：不要动，尽量镇静，让周围人打120，周围没人尽量优先呼叫信任的人，让其帮你拨打120，只需一句：我在家中、快要猝死、拨打120。

  • 心肺复苏

颈动脉负责向大脑供血，出现颈动脉斑块会增加中风的风险

  1. 抗炎，减缓斑块形成。
    • 减肥（肥胖促进血管皮损伤和炎症反应）。
    • 不抽烟：损伤血管内皮，导致脂质成分更容易进入血管壁，引发炎症。
    • 多运动：运动抗炎。
    • 饮食调整：少吃精致的碳水红肉，多吃水果蔬菜全谷物，鱼类坚果和豆制品
  2. 降脂：低密度脂蛋白胆固醇降得越多，斑块逆转越明显。使用他汀类药物，使低密度脂蛋白胆固醇降到1.8以下。
  3. 降压：血压是血流对血管壁的冲击力，高血压会加重血管损伤，尽量控制血压在130-80毫米汞柱以内。
  4. 控糖：高血糖也会引起血管壁的慢性炎症反应，把糖化血红蛋白控制在7.0以下。

心肺复苏CPR

急救CPR（心肺复苏）：

  1. 呼叫病人，判断意识和呼吸，及时拨打120。
  2. 按压双乳中间点，双手叠扣，惯用手在上（左手在下），双臂伸直，以每分钟大于100次（大概1s2下）按压30次。
  3. 30次后检查口鼻是否有异物，抬高下颚，捏住鼻子，嘴包住嘴吹气2次。
  4. 30次按压接2次呼吸，循环。

胃肠

胀气的原因：首先检查胃肠是否有疾病。若没有器质性疾病，则有可能：

  1. 吃饭太快：没有嚼成糊，食物块之间缝隙较大，会导致吞咽进去较多空气。
  2. 少吃含气的食物：比如碳酸饮料、苏打水、啤酒。
  3. 少吃产气的食物：萝卜、黄豆，高纤维食物分解后产生沼气，但纤维食用过少会导致肠道蠕动慢，食物在胃里呆的久也会导致胀气。
  4. 乳糖不耐受：喝牛奶
  5. 不运动
