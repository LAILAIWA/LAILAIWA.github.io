食物归纳

发表于 2022-01-29 更新于 2022-03-15 分类于 编程之外 ， 健康
本文字数： 1.1k 阅读时长 ≈ 1 分钟
整理胃病期间需要注意的食品摄入。

食物归纳

注意：

  • 饮食清淡。
  • 少食多餐，细嚼慢咽。
  • 饭后宜卧床休息1-2小时。保证肝脏得到充足的血液供应，有利于肝细胞修复和再生，帮助恢复肝功能。
  • 尽量采用蒸、煮、炖、氽、熬的烹调方法，少采用煎、炸、炒的烹调方法，坚持少盐饮食。
  • 多饮水。
  • 少熬夜。
  • 减少焦虑。
  • 适量的运动，不宜剧烈运动。

全麦面包 是 高纤维
红米 是 高纤维
芝麻 是 高纤维
花生 是 高纤维
杏仁 是 高纤维
腰果 是 高纤维
核桃 是
绿豆 是

二. 荤菜

名称 食用 备注
鸡蛋 是 胆固醇不能说明有影响，优质蛋白质、维生素、卵磷脂。
瘦肉 是
肥肉 少量 不宜多食
鱼 是 补充蛋白质和维生素，有较好的抗氧化功能且易消化吸收
虾 少量 胆固醇高，优质蛋白，尽量清淡
螃蟹 少量 胆固醇高
动物肝脏 否 胆固醇高
动物脂肪 否 猪油、肥猪肉、黄油、肥羊、肥牛、肥鸭和肥鹅等
动物皮 否 禽类皮
鱼子 否 胆固醇高
鱿鱼 否 胆固醇高

三. 蔬菜

名称 食用 备注
蔬菜 是 补充维生素，有较好的抗氧化功能且易消化吸收
芥蓝 是 高纤维，维生素，蛋白质，钙。
西蓝花 是 高纤维，维生素C
菠菜 适量 高纤维。草酸含量较高，肾炎、肾结石一次不宜多食。
蘑菇 是 高纤维，草菇，冬菇，蘑菇，云耳
大蒜 否 辛辣，有刺激性。可以降低血清胆固醇、甘油三脂及防治动脉粥样硬化，还可以降低血糖。
洋葱 适量 辛辣，有刺激性，避免生吃。适宜高血压、高血脂、动脉硬化等心血管疾病、急慢性肠炎以及消化不良者。
青椒 否 眼疾患者、食管炎、胃肠炎、胃溃疡、痔疮患者应少吃或忌食

八. 水果冷藏

自己会释放乙烯的水果不适合放入冰箱，乙烯是天然催熟剂，冰箱会抑制乙烯生产。

  • 释放：
    • 香蕉
    • 榴莲
    • 牛油果：优质脂肪，脂肪含量很高。
    • 芒果：富含胡萝卜素。
    • 桃子：桃仁有毒。
    • 杏：富含钾和胡萝卜素。苦杏仁有毒。
    • 猕猴桃：富含VC和膳食纤维。
  • 不释放：冷藏或直接吃。
    • 草莓：冷藏2到3天吃完。
    • 葡萄：冷藏一周内吃完。
    • 菠萝：冷藏5到7天内吃完。
    • 荔枝：冷藏2到3天吃完。含糖量高。
    • 樱桃：冷藏2到3天吃完。
    • 枇杷：冷藏两周内吃完。含糖不高，果核有毒。
    • 石榴：冷藏一个月吃完。
    • 橘子：冷藏一周内吃完。
    • 西瓜：切开后立马吃掉。
