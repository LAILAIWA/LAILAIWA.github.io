    1.4 简单处理

    （1）煮鸡蛋

    • 电锅/蒸锅时间区别不大
    • 冷水开始：
      • 蒸蛋：15分钟左右
      • 煮蛋：10分钟左右（未验证）
    • 水烧烤后下蛋：
      • 蒸蛋：10分钟左右
      • 煮蛋：5分钟左右（未验证）
    • 冷水下蛋相比滚水下蛋，鸡蛋因为缓慢受热，最终口感上可能会更嫩滑一些，后者可能会较硬

    （2）蒸米

    • 分量：

      • 米和水比例一般为1:1.5到1:2，如1碗米两碗水，当然不同米可能比例有差别（东北米为1:1.25）。
      • 通常200克米可以煮出两到三碗米饭（每碗100克）

    • 淘米：清水洗两三遍即可。

    • 根据烹饪工具区分：

      1. 电锅：
        • 大火开盖煮10分钟，不要盖盖子可能会导致水溢出。
        • 水份蒸发的差不多后转小火，盖盖子焖煮10分钟。
        • 搅拌一下，锅底的米饭翻到上面，最后小火盖盖子焖煮5分钟。
      2. 炒锅：明火
        • 火力比电锅大，所以大火改为中火
        • 小火调为最小火
        • 步骤与电锅相同
      3. 蒸锅：盖盖子，蒸完关火后最好焖2分钟提升口感
        • 冷水上米蒸大概35~40分钟。
        • 水烧开蒸大概30分钟。
      4. 电饭锅：根据使用说明操作即可。

    • 优化：

    （3）蒸馒头

    【馒头/包子·第1课】鲜酵母VS干酵母，怎么用

    （4）凉菜拌汁

    学会这款万能凉菜拌汁，拌凉菜放上一勺更有味道，荤菜素菜都能用

    法式欧姆蛋

    二. 菜品清单

    2.1 面食

    2.1.1 拌面

    凉拌面

    面条处理方法：

    1. 煮熟面条（碱面），不过凉水（风干放凉，口感更干爽），自然摊开，拌一点熟油防粘。做多一些可以保鲜膜冷藏。
    2. 煮熟面条，捞出过一遍凉开水。

    拌酱：搅拌均匀，加入凉面。

    1. 油泼：
      • 1勺蒜末、2勺辣椒粉、1勺花椒粉、两勺芝麻、1勺葱花，热油呲一下
      • 再加入2勺酱油、1勺耗油、2勺陈醋（适量芥末）
    2. 油泼：
      • 小米辣、芝麻、蒜末、粗细辣椒面，淋入热油
      • 一勺生抽，两勺陈醋，半勺耗油，适量糖，搅拌均匀
      • 加香菜、花生米、黄瓜，倒入酱汁，搅拌均匀
    3. 朝鲜族拌面
    4. 怪味冷面
    5. 番茄凉面

    配菜：

    1. 黄瓜、胡萝卜、生菜、豆芽、豆苗、香菜
    2. 火腿、酱牛肉、花生米
    3. 酸豆角、腌白菜/泡菜。

    食材：方便面、葱、蒜、辣椒面、白芝麻、黄瓜丝、花生米、香菜

    步骤：

    1. 空碗放入蒜末、葱花。辣椒面、熟芝麻。
    2. 淋上热油
    3. 加两勺生抽，两勺陈醋，一勺耗油，少许盐糖，搅拌均匀。
    4. 煮软方便面，加入香菜、黄瓜丝、花生米，搅拌均匀。

    参考：6道日小凉面

    酱拌面
    1. 面条煮熟，过两遍凉水。
    2. 空碗加入蒜末、葱花
    3. 一勺生抽，一勺辣椒油，一勺白糖，一勺白芝麻，一勺花生米，两勺芝麻酱，半勺老抽，两勺开水。
    4. 搅拌均匀
    鸡蛋火腿拌面

    食材：面条，线椒，小米辣，鸡蛋，火腿

    步骤：

    1. 酱汁：两勺生抽，一勺老抽，一勺耗油，一勺黄豆酱，一勺淀粉，半碗清水，搅拌均匀
    2. 面条煮熟
    3. 鸡蛋煎熟
    4. 锅内留油，倒入蒜末辣椒火腿翻炒
    5. 倒入调好的酱汁煮开
    6. 放入鸡蛋面条搅拌均匀
    麻辣拌

    食材：火锅丸子、面条、金针菇蔬菜等麻辣烫配料

    调料：蒜末、葱花、辣椒面、白芝麻

    步骤

    1. 酱汁：空碗放入蒜泥、辣椒面、白芝麻、葱花，淋入热油，放入三勺生抽，一勺醋，一勺耗油，一勺白糖，两勺芝麻酱，少许盐糖，搅拌均匀。
    2. 食材煮熟，沥干水分，放入碗中，倒入酱汁，搅拌均匀。

    2.1.2 蒸面

    蒸卤面

    食材：豆角、五花肉、洋葱、姜、蒜、细面条

    调料：

    步骤：

    1. 水滚后蒸15到20分钟左右面条。
    2. 洋葱、姜、蒜切粒。
    3. 热锅冷油，开中小火爆炒，将洋葱、姜、蒜炒香。
    4. 加入五花肉。
    5. 加适量酱油，豆瓣酱。
    6. 加凉白开、辣椒粉、生抽。
    7. 倒入豆角，煮10分左右，注意不要熬干。
    8. 再加入适量水，足以湿润全部面条，大火煮沸。
    9. 面条倒入锅内，与汤汁拌匀，焖5到6分钟。
    10. 如果面条不够软，再放置蒸锅蒸几分钟。

    2.1.3 炒面

    炒拉条

    食材：瘦肉、洋葱、青椒、绿豆芽、西红柿。

    调料：盐、老抽、味精、辣椒油、孜然粉。

    步骤：

    1. 瘦肉切丝，洋葱和青椒切丝，西红柿切丁，适量绿豆芽作为配菜。
    2. 准备好面条：如拉面。
    3. 滚水下面，利用“水刀”分开面条，煮熟面捞出，浸入冷水。
    4. 热锅冷油，下入配菜翻炒，加入适当老抽调色。
    5. 翻炒几下后，加入少许清水，开始调味。
    6. 加入盐、味精、辣椒油。
    7. 翻炒均匀后，捞出面条，控干水分，下锅翻炒。
    8. 加入适量孜然粉，翻炒均匀。
    9. 出锅，翻炒大火情况下几下即可。

    手撕包菜

    食材：五花肉、包菜

    调料：蒜、干辣椒、花椒、盐、白糖、鸡精

    1. 包菜切半，切除根部，叶子撕成均匀小块，厚的杆子抛弃（可以做泡菜），洗干净，沥干水份
    2. 蒜切片
    3. 热锅少许油，下五花肉炒出油
    4. 倒入蒜片干辣椒花椒，炒出香味
    5. 加入少许料酒
    6. 加入生抽，最好和油炒一下，会更香，不能炒太久，可能会变糊发苦
    7. 包菜下锅，大火翻炒，炒至包菜微微发蔫
    8. 盐、白糖、鸡精调味，炒匀
    9. 出锅前可以放一点醋，翻炒几下出锅
    炒胡萝卜丝

    食材：胡萝卜、青椒

    调料：蒜、干辣椒、花椒、盐、白糖、鸡精

    1. 胡萝卜切丝，加入盐和白醋腌制15分钟，使其炒出来口感变脆，腌后倒掉水份
    2. 青椒切丝、大蒜切片备用
    3. 热锅放油（猪油更香），烧热后放入蒜片炒香
    4. 倒入胡萝卜翻炒
    5. 放入青椒、食盐、少许生抽，翻炒
    6. 炒软后出锅

    2.4 炒饭/拌饭

    肥牛拌饭

    食材：肥牛、洋葱、葱、姜、鸡蛋

    调料：料酒、生抽、老抽、耗油、盐糖、胡椒粉、淀粉

    温泉蛋：

    1. 1000ml水烧开关火，加300ml凉水。
    2. 放入无菌蛋焖10分钟。
    3. 捞出后泡几分钟冰水
    1. 冷水加葱姜、料酒，煮开后放入肥牛焯变色捞出。
    2. 调料汁：2勺料酒、2勺生抽、半勺老抽、1勺耗油、1勺盐、1勺糖、适量胡椒粉、1勺淀粉，加一碗清水搅匀
    3. 半个洋葱切丝。
    4. 热锅冷油，先将洋葱炒出香味。
    5. 加入肥牛，倒入料汁，小火煮至浓稠即可。
    6. 米饭上加入肥牛和温泉蛋，浇上酱汁，撒上芝麻和葱花。

    2.5 炖菜

    水煮牛肉

    食材：牛肉、葱姜蒜、鸡蛋、黄豆芽

    调料：胡椒粉、辣椒面、盐、淀粉、食用油、生抽、老抽、耗油、花椒、豆瓣酱、火锅底料、芝麻

    步骤：

    1. 牛肉逆纹路切薄片，泡冷水洗去血水。
    2. 加入少许葱姜水、鸡蛋清、胡椒粉、盐、淀粉，抓匀后封一层食用油。
    3. 热锅冷油，将葱姜蒜、花椒炒香。
    4. 加一勺豆瓣酱、一块火锅底料炒化。
    5. 加水煮5分钟，再加入2勺生抽、1勺老抽、1勺耗油、1勺盐。
    6. 捞出料渣，加入豆芽烫十几秒就捞出，放入盆底。
    7. 牛肉下锅几秒后再轻轻翻动，1分钟后可以捞出，铺在豆芽上。
    8. 再在盆中加一些汤汁，上面撒上蒜末、花椒、芝麻、辣椒面、葱花，淋上热油即可。

    参考：水煮牛肉

    番茄炖牛肉

    食材：牛肉、番茄、土豆、蒜、洋葱

    调料：盐、生抽、耗油、糖

    步骤：

    1. 冷冻的番茄，温水泡一会，方便去皮，切块
    2. 牛肉煎至表面金黄，煎出油脂
    3. 热油爆香蒜末和洋葱丁
    4. 放入番茄炒出汁（炒成汁可以不加水）
    5. 倒入牛肉，加入生抽、耗油、糖
    6. 小火加盖炖70分钟
    7. 加入土豆炖15分钟
    8. 出锅前再放入番茄块，加盐调味

    西红柿炖牛肉，别直接下锅炖，学正确做法，汤鲜味浓，肉香软烂

    鸡公煲

    食材：鸡腿、鸡翅根、土豆、洋葱、芹菜、千张、丸子、方便面

    调料：葱姜蒜、豆瓣酱、黄豆酱、生抽、耗油、白糖、花椒粉、盐、干辣椒

    步骤：

    1. 起锅烧油，下入鸡腿肉炒出水份
    2. 放一勺豆瓣酱炒出红油
    3. 下入葱姜蒜炒香
    4. 放两大勺黄豆酱炒匀
    5. 放入两勺生抽、两勺耗油、三勺白糖、适量花椒粉和盐
    6. 加入热水没过鸡肉
    7. 再加入半瓶啤酒
    8. 放入土豆块和干辣椒
    9. 盖盖小火煮15分钟
    10. 放入丸子类继续煮10分钟
    11. 最后加入如千张、洋葱、方便面、芹菜等好熟的食材，盖盖煮5分钟即可。
    12. 喜欢干锅再大火收汁

    2.8 点心

    韭菜馅饼、韭菜盒子

    食材：面粉，韭菜，鸡蛋，（虾皮），盐，胡椒粉，耗油。

    揉面：

    1. 400克面粉 + 1小勺盐，搅拌均匀。
    2. 140ml开水烫一半的面，另一半用140ml凉水和面。
    3. 两种面搅拌均匀，用手揉成团。
    4. 在硅胶垫上搓衣服一样揉搓4~5分钟（不带手套会比较粘手）。
    5. 硅胶垫上撒一层干面粉，将面团搓成长条，均匀切断，揉成一个个圆球。
    6. 在圆球表面涂一层薄油，锁住水分，盖上保鲜膜，醒20分钟以上，最后面会比较延展。
    1. 准备干净的韭菜（600克）。
    2. 打6个鸡蛋，搅拌均匀，温油倒入，并快速划散，炒成小颗粒。
    3. 韭菜，切去根部，再切丁，加入适量油，搅拌均匀。
    4. 加入放凉的鸡蛋，（一把虾皮），适量盐和胡椒粉，以及耗油等，搅拌均匀。
    5. 将面按大按薄，加入馅料，并压实，捏合两侧面皮，最后留尾部一块盖住整个底部。
    6. 底部朝下，摔一下摔平。
    7. 电饼铛微热，刷一层油，加入一个个包好的面团，并按薄，顶部再喷一层油，中火+盖盖子。
    8. 烙至底部金黄翻面。
    9. 另一面金黄再翻一次。
    10. 两面金黄熟透出锅。

    参考：韭菜馅饼

    韭菜盒子这样调馅才更香，再学会和面烙饼小窍门，皮薄不破更解馋

    酱香饼

    食材：面粉，芝麻，葱花，洋葱，香菇（大蒜、生姜）

    调料：豆瓣酱、花椒粉、五香粉、糖、辣椒面（甜面酱、黄豆酱、豆腐乳、芝麻酱）

    1. 300克面粉，3克盐，搅拌均匀。
    2. 加入210ml，50~60度的水，搅成絮状。
    3. 移至揉面垫上，用手掌位置推拉揉搓面团，揉至光滑。
    4. 表面喷油，搓成长条，分成两份，分别揉圆。
    5. 盖上保鲜膜，醒15分钟。
    6. 取小碗，加入10克面粉，15克食用油，搅拌均匀，调成油酥。
    7. 揉面垫喷一层油防粘，取出面团按扁，擀成圆型。
    8. 倒上油酥，涂抹均匀。
    9. 切成四份，其中三份再分成两份，只切一半，不彻底切开。
    10. 然后把6份小面片摞起来，留剩下的一份大面片，边缘按薄，用大面片包住小面片，整理成圆形面坯。
    11. 另一份也如上述操作，盖上保鲜膜，醒10分钟。
    12. 醒完擀成长方形饼坯。
    13. 电饼铛微热，刷多点油，饼浸入油里炸，卷起饼坯，慢慢放入锅中，一边放一边推出褶皱。
    14. 表面也刷一层油，大火+盖盖子。
    15. 烙至底部金黄表面变色翻面。
    16. 另一面金黄再翻一次。
    17. 涂上一层酱香饼调味酱，以及熟芝麻和小葱花

    调味酱（版本1）：

    1. 半个洋葱切碎，几个香菇切碎。
    2. 200g热油炒香碎洋葱，炒至焦黄发干。
    3. 下豆瓣酱炒出红油（加一小碗肉末）。
    4. 加入番茄酱和香菇末。
    5. 撒点花椒粉和五香粉，加入适量清水。
    6. 水煮开后加入适量白糖和辣椒面，熬至黏糊。

    （版本2）

    1. 半个洋葱切块，5瓣大蒜，适量生姜和葱白，2勺豆瓣酱，1勺甜面酱，1勺黄豆酱，1块豆腐乳，1小勺五香粉，1小碗清水，料理机打碎。
    2. 热油倒入酱料，中小火炒出香味，转小火焖至水分收干一些。
    3. 加一勺白糖，熟芝麻（芝麻酱），转大火炒至稀稠。
    4. 装入密封的保鲜盒，用不完可以冷藏。

    参考：酱香饼、酱香饼、酱香饼

    蜂蜜小面包

    食材：

    • 高筋面粉（350克）、低筋面粉（50克），燕子酵母（5克）、奶粉（25克）
    • 糖（30克）、盐（4克）、鸡蛋（1个）
    • 奶油（80克）、牛奶（175克）、蜂蜜（40克）
    • 黄油（20克）、白芝麻（20克）、白砂糖（30克）
    • 玉米油

    器材：空气炸锅

    1. 将黄油上面的食材均倒入桶内，搅拌均匀，液体后放。
    2. 揉至可以拉出厚膜。
    3. 加入软化的黄油，揉至可以拉出薄膜。
    4. 团成面团密封发酵到2倍大小。
    5. 按压，挤出空气，切成均匀的8份，整理成圆形。
    6. 盖上保鲜膜，醒面15分钟。
    7. 白芝麻和白砂糖混合，搅拌均匀。
    8. 烤盘倒入一些玉米油，加入少量黄油增加奶香。
    9. 面团取出擀开，两边向中间折叠，再次擀开，两头压平，卷起后从中间切开。
    10. 底部沾满糖芝麻，摆进烤盘。

    2.9 汤类

    绿豆沙

    步骤：

    1. 先将绿豆淘洗干净，放入开水中浸泡。
    2. 待开水变凉后，保留少量水，放入冰箱中冷冻结块。
    3. 锅内放置少量纯净水（1小块陈皮），大火烧开。
    4. 热水下入冷冻的绿豆，大火煮散。
    5. 待再次烧开后，转中小火，盖盖煮10分钟，此时绿豆变沙。
    6. 加入适量水（放入冰糖），大火把水烧开，撇下浮沫。
    7. 可以晾凉后放入冰箱冷藏。

    2.10 腌制

    腊八蒜/糖蒜

    材料：

    • 首选紫皮蒜，新蒜好于老蒜。夏天的新蒜要先冷藏一下。大批量去蒜皮，温水浸泡1小时。切掉蒜根绿的快一些。
    • 米醋（颜色好看），老陈醋（口感好一些，推荐宁化府）
    • 冰糖
    • 密封瓶，避免金属瓶盖

    步骤：

    1. 锅中加入500g醋，50g冰糖 或细砂糖，开火煮开（使醋味柔和）。
    2. 晾凉等冰糖化开，糖中和醋酸（也可以不加）。
    3. 把剥好的大蒜放入瓶子，倒入晾凉的醋汁淹没大蒜。
    4. 放入冰箱冷藏（14天）/冬天放在户外避光（7天），温度越高绿的越快，但口感会差一些。

    注意：

    1. 腊八蒜变绿后要马上放入冰箱，并尽快吃完，不然大蒜会慢慢变软。
    2. 先低温冷藏过再处理的大蒜容易变绿
    3. 低温，醋酸，避光
    4. 如果产生气泡说明温度较高，口感会稍酸
