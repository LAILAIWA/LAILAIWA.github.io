高效学习方法是否靠谱

发表于 2023-02-27 更新于 2023-04-04 分类于 笔记杂谈
本文字数： 3.3k 阅读时长 ≈ 3 分钟
评价一下某些高效学习方法，是否能应用于日常学习。

高效学习方法是否靠谱

互联网上常有所谓的“高效学习方法”，但其实这些方法是否真的能提高学习效率，有待商榷。是否存在一套通用的学习方法适用于每个人，还是说学习是因人而异？我和大牛之间是不是只是少了一套武功秘籍？

我觉得答案是否定的，不同的知识很难用一个大一统的理论来囊括出一个最优解，但学习可以有好习惯和坏习惯。网上流传的学习方法是否真的高效，需要经过自我评估后，才能确定其是否对学习有实质的帮助。

一. 番茄工作法

一种时间管理方法，内容：

  • 选择一个任务，专注工作25分钟，然后修复5分钟，一个番茄时间为30分钟，不可切分。
  • 每4个番茄时间后休息30分钟。
  • 每完成一个任务划掉一个任务，每个番茄钟未完成需要重新计时。
  • 耗时超过6个番茄钟的任务要再切分，短任务之间可以合并进一个番茄钟。

从内容看，该方法主要有几个特点：

  • 以一个番茄钟为最小单位，是大部分人能够保持专注的一个时间范围，这意味着每次面临的任务难度都尚可。
  • 拆分任务的过程意味着要对学习进行计划，把一个复杂的任务，比如学习编程，学习一门外语，划分为读哪些书、看哪些教程、做哪些练习，比如刷1000道题，背3000个单词，把一个看起来难度很高的任务拆分为当前一个个能完成的小任务。
  • 固定的学习和休息节奏，让自己能够按制定的计划有条不紊的完成自己的当日目标，保持高质量高集中力的学习时间，从长期来看相比漫无目的的学习也更容易坚持下去。

总结优点：

  1. 拆分任务，复杂问题简单化。
  2. 劳逸结合，降低疲劳感。
  3. 有计划，容易实施和坚持。
  4. 过程有节奏感，有趣味性。

二. 费曼学习法

首先费曼学习法不是费曼提出的学习方法，甚至可以说和费曼毫无关系，从互联网上可以搜集的信息来看，该方法来自畅销书《如何高效学习》。

内容：

  1. 选择一个想要学习的概念，写下它的名称。
  2. 用自己语言向一个初学者解释这个概念，尽量简洁清晰。
  3. 如果在解释过程遇到困难或模糊，就回顾原始资料，直到能顺畅解释。
  4. 简化语言和例子，去掉多余的术语和细节，使解释更加通俗易懂。

从内容来看，所谓的费曼学习法并没有指导作为初学者的我们如何快速掌握知识，而是专注review阶段。

这对于大部分人来说并没有太大意义。因为你很难找到一个现实世界输出的目标，而使用假想目标时又没办法确认自己是否真的做到了简化和清晰的要求。相比于能够编程实现的番茄工作法，所谓的费曼学习法不能算是一个具体能够执行的方法。

优点：

  1. 从被动记忆转向主动理解。
  2. 通过输出内容和简化来巩固和检查自己的输入。
  3. 强迫自己不要忽略学习过程的模糊点。
  4. 简化和清晰表达就是做减法，

三. 西蒙学习法

内容：“对于一个有一定基础的人，只要肯下功夫，可以在6个月内掌握任何一门学问”。1个人可以1分钟到1分半来记忆一个信息，粗略的估计一门学科大致包含5万个信息，需要834个小时，若每周学习40个小时，需要20周也就是5~6个月。

就其内容而言，所谓的西蒙学习法不能称作一种学习方法。首先，我们很难理解它所描述的信息单位是什么以及如何去衡量。其次，不同的知识难度不同，对于不同的人来说，学习速度和质量也会有所不同。举个例子，一个学霸可能只需要高效学习一个小时就能掌握普通人一天的学习量。

所以该方法没有任何现实指导意义。但我们可以相信它的观点，如果你没办法在6个月内学好一门学问，那一定是自己不够努力，而不是你自身智力、体力、专注度或者记忆力上的缺陷，这有助于我们提高自信心。

四. 思维导图

思维导图，一个帮助组织和串联信息的视觉工具，协助我们总结、记忆和思考。如何将思维导图运用于我们的日常学习中？

首先我们分析思维导图的优点：

  1. 图形化：将信息和想法可视化表现，更容易理解和记忆复杂的概念。
  2. 非线性结构：区别于传统笔记的线性结构，能够更加整体的串联知识点。
  3. 组织信息：以一种有逻辑的方式来组织信息和想法，更容易看到信息间的关系。
  4. 创造性：其形式更容易看到各种概念间的联系，从而帮助我们产生新的想法。
  5. 记忆力：图形的形式和信息间的串联更有助于大脑记忆。
  6. 沟通：方便人与人之间的沟通，减少不必要的信息，提供简洁清晰的主题概述。

如何将以上优点运用在我们的学习上，我们以学英语举例，英语要学习如下内容：

  1. 词汇：通过主题对单词进行分类，比如同义反义、派生、特殊变形等，更容易明白单词间的联系，方便记忆。
  2. 语法：对句子进行拆分，比如主语、动词、宾语等，方便我们理解句子构成。
  3. 写作：协助我们整理思路，罗列我们要描述的主题，我们的论点和论据等等。
  4. 阅读：总结文章和段落的主要内容和观点，以及支持观点的细节，提高我们对文章的理解程度。
  5. 口语：选择一个要说的话题，从头脑风暴开始，将你与话题相关的想法用单词或短语写下并连接起来；整理你的想法，并进行逻辑分类和排序；为每个想法增加细节；按你的思维导图进行演说练习，录下你的演讲，反复听和修改，不断优化。
  6. ……不再赘述。

综上所述，与其说思维导图是一种学习方法，不如更准确地说它是一种学习辅助工具。

五. 一些细分知识下的学习技巧

5.1 影子跟读（shadowing）

有助于口语，加强发音、语调和语速方面的练习，也可以提高听力理解能力。

  1. 找到一段适合自己当前英语水平的音频，可以是短对话、听力素材、有声新闻、影视剧片段等。
  2. 播放时注意听清发音和语调，并理解意思。
  3. 播放后马上跟读这段内容，尽量模仿原音。
  4. 反复进行跟读练习，多次练习同一段音频，直到自己可以流利、自然的说出为止。

5.2 自然拼读法（Phonics）

自然拼读法是指看到一个英语单词，就可以根据英文字母在单词里的发音规律把这个单词念出来的一种方法。它通过建立字母或字母组合与词形之间的对应关系，即词汇的各种拼读规律，帮助学生读、写和记忆词汇。

儿童学习语言的顺序是听、说、读、写，积累了一定量发音的词汇量后并不意味着能够“识字”，自然拼读法是串联发音和单词拼写的桥梁。但中国的英语教育中儿童是从阅读开始，没有前置的词汇积累阶段。

步骤：

  1. Decoding
  2. Reading Fluency
  3. Reading Automaticity
  4. Reading Comprehension

该方法的最佳使用场景应该是有语言环境，在练习阅读前就通过口语练习积累了一定量词汇的儿童。但即使我们不满足这种条件，做一些类似的专项练习，掌握一些常见的拼读规则，对于补充词汇也是有帮助的。

六. 学习习惯

6.1 早起毁一天 vs 早起高效学习？

  • 早起毁一天：认为为了早起会导致睡眠不足，降低白天的精力。
  • 早起高效学习：认为早起（比如5点）是一天精力最好的时候，没有干扰，可以利用早饭前的两个小时来高效的学习。

首先根据早起会导致睡眠不足的观点，早睡可以解决睡眠不足的问题。实际上每个人一天的时间都是固定的24小时，假设我们都没有短眠基因，每天需要8个小时的睡眠，如果时间都是一样的，那早起的意义在哪里？认为早起更好的我们是否和朝三暮四的猴子一样？

早起的优点：

  • 夜晚时人更容易懈怠，忙碌一天后想要休息和娱乐的欲望会更强。
  • 早上比夜晚更安静，没有现实和网络世界的噪音。
  • 早上精神更饱满，思维更敏捷。

如果你认同上述优点，早起对你就是有意义的。如果你不认同，比如你认为就算睡了8小时，你凌晨还是会犯困（可能是短期生物钟调整带来的疲惫感），或者你有一个吵闹的室友让你无法早睡，那么早起对你就没有价值。

6.2 越阶挑战

不是所有人都能越挫越勇，what Doesn’t Kill You Makes You Stronger，it can also make you weaker。痛苦是生命中不可避免的一部分，我们赋予了它变革的力量来让创伤给我们带来积极的意义，比如没有被抗生素杀死的细菌会发生变异，从而产生了抵抗力。但人并非细菌，灾难中幸存下的人也并不能因为灾难而获得力量。

在学习中，盲目的给自己设定高标准只会带来失败后的挫败感，这些挫败感对学习没有任何帮助，也不会因为越阶挑战成功而提高效率，只能说明你没把控好当前的学习进度。一直保持高昂的学习兴趣和自信心，稳步的提升难度是普通人应该坚定的路线。

6.3 归纳

只作简单罗列，顺序与重要性无关。

  • 制订计划，分解任务
  • 创造学习环境
  • 多种多样的学习方式
  • 避免低质量的学习材料
  • 下定决心，反复练习
  • 摒弃高压，稳步提升，重在坚持

高效学习方法并不高效，认真的按计划学习也不低效。
